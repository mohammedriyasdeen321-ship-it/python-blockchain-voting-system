# 🗳️ Blockchain-Based Voting System

A secure web-based voting application built using **Python** and **Flask**, implementing blockchain concepts to ensure transparency and integrity in the voting process.

---

## 🚀 Features

- 👤 User Registration & Login
- 🗳️ Secure Vote Casting
- 🔒 Blockchain-Based Vote Storage
- 👨‍💼 Admin Dashboard
- 📊 Live Election Results
- ✅ One Vote Per User
- 💾 SQLite Database

---
